# RayTracingInOneWeekend

### 光线追踪实践

使用了miloyip的极简PNG编码函数svpng[svpng](https://github.com/miloyip/svpng "svpng")

# Usage

~~~
cd project
mkdir build
cd build
cmake ..
make
~~~
